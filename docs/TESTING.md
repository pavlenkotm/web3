# Testing Guide

## Solidity Tests

```bash
npx hardhat test
npx hardhat coverage
```

## Python Tests

```bash
cd python/web3-cli
pytest
```

## Rust Tests

```bash
cd rust/solana-token-program
cargo test
```

## Go Tests

```bash
cd go/eth-rpc-client
go test
```

## Test Coverage

Aim for >80% code coverage across all projects.
